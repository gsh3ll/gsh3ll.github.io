# 0x00 什么是 John the ripper

John the Ripper 是一个快速的密码破解工具，用于在已知密文的情况下尝试破解出明文的破解密码软件，适用于 Windows 和许多 Linux 系统，支持目前大多数的加密算法，譬如 DES、MD4、MD5，其主要目的是破解不够牢固的 Unix、Linux 系统密码。支持字典破解和暴力破解。

# 0x01 如何使用

john the ripper 的执行文件在 /usr/sbin/john 目录下，其自带的字典文件在 /usr/share/john 目录下。

破解 Linux 密码需要用到两个文件：

  1. /etc/passwd 包含用户信息的文件
  2. /etc/shadow 包含密码信息的文件
