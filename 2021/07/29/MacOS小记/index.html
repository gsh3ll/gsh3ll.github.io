# Java 环境共存

# 方法一

java 会安装在这个目录： /Library/Java/JavaVirtualMachines/

先安装最新版本的 java，然后安装 java8，并使用下面命令把 java8 链接到 java8

# 方法二

~/.bash_profile  文件如下：

#java
export JAVA_15_HOME=/Library/Java/JavaVirtualMachines/jdk-15.0.2.jdk/Contents/Home
export JAVA_8_HOME=/Library/Java/JavaVirtualMachines/jdk1.8.0_281.jdk/Contents/Home
# 默认为JDK15
JAVA_HOME=$JAVA_15_HOME

alias jdk15="export JAVA_HOME=$JAVA_15_HOME"
alias jdk8="export JAVA_HOME=$JAVA_8_HOME"

切换版本时使用 jdk15  或者 jdk8  即可。

# 方法三

编辑 /Users/xxx/.zshrc ：

#### 切换jdk版本的函数
jdk(){
 echo "
 >> Usage: jdk  [version]
  "
 version=$1
 unset JAVA_HOME;
 export JAVA_HOME=$(/usr/libexec/java_home -v "$version");
 #export CLASS_PATH="$JAVA_HOME/lib"
 #export PATH=".$PATH:$JAVA_HOME/bin"
 java -version
}

切换版本时使用 jdk15 或者 jdk8 即可。

# 安装插件

  • 快捷键 ctrl+shift+P ，或在工具栏里依次点击 Preferences -> Package Control 。

  • 在弹出的命令框里输入 Install ，找到 Package Control: Install Package ，回车。

  • 输入要搜索下载的插件名，回车即开始下载安装插件。
