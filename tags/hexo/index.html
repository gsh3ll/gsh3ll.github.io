搭建Github-Hexo博客

Hexo是一个基于node.js开发，开源的静态博客生成器。此文章用作记录搭建该博客的过程，以及中途遇到的问题。 因为已经弃用七牛云的缘故，本文的图片已经无法显示，故图片已被删除。 搭建准备 Hexo是一个开源的静态博客生成器，用node.js开发，所以在安装Hexo前我们需要先安装node.js。 进入node.js主页后下载LTS版本，双击安装即可。安装完成后可通过下面命令查看版本信息。 gsh3ll:~ $ node -v ##查看node版本 v12.15.0 gsh3ll:~ $ npm -v ##查看npm版本 6.13.4 通过npm来安装Hexo，但是受国内环境影响，下载速度很慢。可通过安装cnpm来解决。 gsh3ll:~ $ npm install -g cnpm --registry=https://registry.npm.taobao.org gsh3ll:~ $ cnpm -v ##查看cnpm版本 cnpm@6.1.1 (/usr/local/lib/node_modules/cnpm/lib/parse_argv.js) npm@6.13.7 (/usr/local/lib/node_modules/cnpm/node_modules/npm/lib/npm.js) node@12.15.0 (/usr/local/bin/node) npminstall@3.27.0 (/usr/local/lib/node_modules/cnpm/node_modules/npminstall/lib/index.js) prefix=/usr/local darwin x64 19.2.0 registry=https://r.npm.taobao.org 最后使用cnpm安装Hexo即可。 gsh3ll:~ $ cnpm install -g hexo-cli ##安装Hexo gsh3ll:~ $ hexo -v ##查看Hexo版本 hexo-cli: 3.1.0 os: Darwin 19.2.0 darwin x64 node: 12.15.0 v8: 7.7.299.13-node.16 ... unicode: 12....
